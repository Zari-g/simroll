# SimRoll — Project Requirements

## Vision

SimRoll is a playful Brazilian Jiu-Jitsu simulation platform for exploring positions, transitions, submissions, escapes, and gi/no-gi grappling pathways.

The project aims to create an interactive grappling sandbox where users can simulate rolls, explore pathways between positions, and experiment with different grappling systems.

## Core Concept

The system models Brazilian Jiu-Jitsu as a graph:

- positions are nodes
- transitions are edges
- grips act as constraints
- pathways represent possible sequences during a roll

## MVP Goals

The first version should support:

- positions
- transitions
- gi/no-gi tagging
- grip logic
- pathfinding between positions
- basic querying of available techniques

## Long-Term Goals

Future versions may include:

- interactive website
- mobile app
- roll simulation
- visual position maps
- saved pathways
- user-generated systems
- funny/chaotic BJJ interactions
- AI-assisted exploration
